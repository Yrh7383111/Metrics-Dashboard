# Metrics Dashboard


## Project Setup
```
pod update
```


## Development Stack
```
Front-end: UIKit, SwiftUI

Database: Firebase

Back-end: Swift and other CocoaPods packages

Features: Signup, Facebook Login, Date Range, Analytical Dashboards, Task Planning, Notifications
```
